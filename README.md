# Sales_Dashboard-EXCELR-

The **Sales Insights Dashboard** is an interactive Business Intelligence project built using Power BI. It analyzes sales performance, revenue trends, profitability, and customer segmentation to deliver actionable insights for better decision-making.  

This project transforms raw transactional data into meaningful visual insights using KPIs, charts, and interactive filters.  

---

## 🎯 Objectives  
- Analyze overall sales and profit performance  
- Track monthly revenue, profit, and order trends  
- Compare category-wise sales contributions  
- Understand customer segment distribution  
- Evaluate the impact of discounts on profitability  
- Identify top-performing and underperforming products  
- Enable interactive, data-driven decision-making  

---

## 🛠️ Tools & Technologies  
- **Power BI Desktop** – Data visualization and dashboard creation  
- **DAX (Data Analysis Expressions)** – Measures and calculations  
- **Microsoft Excel** – Data source  
- **Bookmarks & Buttons** – Navigation  
- **Conditional Formatting** – KPI indicators  

---

## 📂 Dataset  
The dataset contains transactional sales data with the following fields:  
- Order ID  
- Order Date  
- Customer Name  
- Product Name  
- Category & Sub-Category  
- Region  
- Segment  
- Sales, Profit, Quantity, Discount  

---

## 🔄 Data Preparation  
- Imported dataset into Power BI  
- Cleaned missing/null values  
- Converted date formats  
- Created Month-Year column for trend analysis  
- Built relationships (if required)  
- Developed calculated measures using DAX  

---

## 📈 Key Metrics (DAX Measures)  

### Basic KPIs  
- Total Revenue  
- Total Profit  
- Total Orders  

### Time Intelligence  
- Last Month Revenue  
- Last Month Profit  
- Last Month Orders  

### Growth Metrics  
- Revenue Change (%)  
- Profit Change (%)  
- Orders Change (%)  

### Additional Metrics  
- Profit Margin  
- Average Discount

## 📊 Dashboard Pages  

### 1️⃣ Overview  
- KPI Cards (Revenue, Orders, Profit)  
- Monthly Trend (Line Chart)  
- Category Comparison  
- Customer Segment Distribution  

**Purpose:** Quick snapshot of overall performance  

---

### 2️⃣ Analysis  
- Profit Contribution (Waterfall Chart)  
- Revenue by Region (Map)  
- Profit vs Discount (Scatter Plot)  
- Sales Funnel  

**Purpose:** Understand performance drivers  

---

### 3️⃣ Details  
- Top 10 Products by Profit  
- Transaction-Level Data Table  
- Revenue Breakdown Matrix  

**Purpose:** Deep dive into granular data  

---

## 🎛️ Interactive Features  
- Date, Region, Category, and Segment slicers  
- Reset button  
- Page navigation buttons  
- Dynamic KPI indicators (▲ ▼ with color coding)  

---

## 💡 Key Insights  
- Technology category generates the highest revenue  
- Discounts can negatively impact profit margins  
- Corporate segment contributes significantly to revenue  
- A small number of products drive most profits  
- Some transactions result in losses despite high sales  
- Monthly trends show performance fluctuations

This dashboard effectively converts raw sales data into actionable insights, helping businesses monitor performance, identify trends, and make informed decisions.
